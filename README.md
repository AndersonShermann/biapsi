# Website Profissional - Ana Beatriz Arêas (Psicóloga Clínica)

Este repositório contém o código-fonte do site institucional da **Psicóloga Ana Beatriz Arêas (CRP 05/72707)**. O projeto foi desenvolvido com foco em acessibilidade, design acolhedor (UX) e performance, servindo como o principal ponto de contato para agendamentos de psicoterapia online.

---

## 🚀 Tecnologias Utilizadas

Para garantir um carregamento rápido e uma manutenção simplificada, o site foi construído com:

*   **HTML5 Semântico:** Estruturação focada em SEO e acessibilidade.
*   **CSS3 Customizado:** Uso de variáveis CSS (`:root`) para consistência da paleta de cores e design responsivo.
*   **Google Fonts:** Integração das fontes *Playfair Display* para títulos e *Open Sans* para o corpo do texto.
*   **Integração WhatsApp:** Fluxo direto para agendamento via API do WhatsApp.

---

## 🎨 Design e UX (User Experience)

O projeto visual foi desenvolvido para transmitir segurança e tranquilidade, elementos essenciais na área da saúde mental.

*   **Paleta de Cores:** Tons de creme (`#fdfaf5`) e verde oliva (`#7e8761`), buscando uma estética orgânica e profissional.
*   **Responsividade:** Layout adaptável para dispositivos móveis, garantindo que o usuário consiga agendar consultas de qualquer lugar.
*   **SEO:** Otimização de meta-descrições e títulos para termos como *Psicoterapia Online*, *Burnout* e *Ansiedade*.
